# 腰椎管狹窄症
隨著年齡增長，腰椎管狹窄症的患病率逐年上升。根據研究，2021年香港大學的一項研究對2385名公開招募的受試者進行了腰骶部（L1-S1）的MRI檢查，確定其多節段發展性脊柱狹窄（DSS）的流行率為7.3%。結果顯示，這些患者可能面臨多處脊柱管壓迫的風險。這種狀況的普遍性不容忽視，因其不僅引發劇烈的疼痛，還大幅降低患者的生活質量。

# 症狀
腰椎管狹窄症的症狀通常包含向腿部放射的疼痛、麻木和肌肉無力，這些症狀在長時間站立或走路時加劇，而坐下或前傾則會有所改善。患者的日常活動受到限制，甚至可能出現神經性跛行，這使得他們在生活中的行動能力受到影響。

# 病因
腰椎狹窄主要分為先天性和後天性。先天性因素包括特發性和軟骨發育不全，伴隨椎根短厚導致中央管狹窄。後天性因素更為常見，主要是退化性椎間盤疾病，隨著年齡增長出現椎間盤退化、小關節及韌帶肥厚、骨刺形成等變化。脊柱滑脫和骨關節炎也為常見原因。此狹窄最常影響 L4-L5 椎節，並與年齡相關的退化變化密切相關。

# 物理治療運動有助復康

## 仰臥腰部旋轉伸展
仰臥，雙手向兩側伸展呈『T』字型。抬起一側膝蓋，使其與髖關節對齊。用對側的手，將膝蓋輕輕向地面方向下壓。保持雙肩緊貼地面，不要離地。將頭部轉向膝蓋的相反方向。

停留30秒, 3次一組, 每天3-4組。

## 屈膝抱胸伸展運動
單膝抱膝：平躺，雙手抱住一側膝蓋，緩慢向胸部靠攏。 

停留30秒, 3次一組, 每天3-4組。
{: .notice--info}

## 神經拉伸運動
仰臥, 用一條大毛巾放係腳踝, 然後用手和毛巾將有坐骨神經痛的一邊直腿舉起，直到腰部或大腿感到疼痛或放射性症狀，之後放下大腿，將頭抬起，然後放低, 重覆做。

10次一組, 每天3組。
{: .notice--info}

## 針對改善核心肌肉力量 (動作控制運動 Motor Control Exercise)

強化臀大肌時，我們在地墊上訓練，利用彈力帶放在膝蓋上方，雙膝彎曲，雙腳平放於地面，與臀部同寬。將一條腿完全伸直（呈45度角）或抬起至平板位置。用你該側腳跟的力量將臀部抬起，直到肩膀、臀部和膝蓋成一條直線。

在最上方收緊臀肌保持10秒，20次一組，每日3至4組。
{: .notice--info}

最後是側身平板運動：側躺在運動墊上，雙腿伸直，雙腳重疊在一起。將你的支撐肘直接放在肩膀下方，確保前臂平放於地面，並指向身體外側。通過將肚臍向脊椎靠攏來啟動核心肌肉，你的身體應該形成一條從頭到腳完美的直線。保持頭部和頸部中立，目光直視前方。

維持這個姿勢10秒，每邊5次一組，每天3至4組。
{: .notice--info}

# 家居熱療法 
   在伸展之前使用熱敷(不要太高溫，以免燙傷)可以提高效果: 10-15分鐘, 每天2-3次。

# 坐姿調整
建議患者避免長時間坐著，每15-30分鐘起身走動，並進行定期伸展運動，以顯著減少症狀。利用毛巾來支撐下背部，承托腰椎的生理孤度，減少腰部壓力
